Readings

Truth Tables Reading

A truth table shows every possible input-output combination of a given function. If a logic circuit has n-number of inputs, there will be 2n possible inputs. The typical truth table is a canonical truth table, a truth table in which the binary code inputs are in ascending order. The output(s) of a truth table depends on the circuit; in general, 1 is High and 0 is Low.
