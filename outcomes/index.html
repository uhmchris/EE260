Learning Outcomes What you will know

This page collects together all of the “outcomes” associated with individual modules. Outcomes identify what students will know and be able to do if they master the material.

Verilog Basics Outcome

  • Understand Verilog Syntax
  • Design and use Verilog modules

Referencing modules: Verilog Basics

Assessed by: Verilog Quiz 1, Verilog Project 1

ASM Charts

  • To be able to recognize an ASM chart
  • Knowing the difference between an ASM chart created using mealy or moore
  • Understanding the value in an ASM chart.

Referencing modules: ASM Charts

Min and Max Terms

  • To learn what minterms and maxterms are
    • To learn about Product of Sums (CPOS) and Sum of Products (CSOP)
  • To be able to recognize the difference between them
  • To learn how to find the min/max terms from a truth table
  • To learn of min/max term uses

Referencing modules: Min Max Terms

Logic Gates

  • To learn about Not, And, Or, Nand, Nor, Xor, and Xnor logic gates
  • To learn about the above logic gates’ circuit simulation symbols
  • To learn about the above logic gates’ boolean algebra symbols
  • To learn about the above logic gates’ truth tables

Referencing modules: Logic Gates

Truth Tables

  • To learn what a truth table is
  • To be able to create a truth table from a given situation
  • To be able to interpret and use a truth table

Referencing modules: Truth Tables
